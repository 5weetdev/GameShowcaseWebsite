Fall Survivor

Fall Survivor is a survival game set in a bizarre world of continuous free fall, where physics behave unpredictably. After escaping an enemy flying ship, the protagonist must survive amidst intelligent creatures, build a base for defense, and manage essential resources like food and water. The quest to uncover the truth behind this strange, falling world drives the adventure.

ASCII Shader

Inspired by Acerola's video on ASCII art rendering, this shader project aims to recreate the iconic Minecraft visual style using ASCII characters. By translating the game's visuals into ASCII art, the shader produces a unique and nostalgic aesthetic.
